# release-202511xx

Release type: Multiplayer early access

Release post: [The first public release of OpenE2140]({% link news/_posts/2025-11-09-first-public-release.md %})

## What's new
- Multiplayer early access: play online or on LAN with other people using our dedicated servers (located in Europe).
- Modernized UI and quality‑of‑life features inspired by the C&C series: unit production queues, unit stances, HCU‑M automatic repair, and automatic ore transport.
- Redesigned visuals and improved controls to make gameplay more accessible and familiar to modern RTS players.

## Known issues
- No playable AI in Skirmish: Skirmish mode is available, but there is no AI opponent yet.
- Single‑player campaigns not implemented.
- Occasional crashes or visual glitches may occur in some situations.
- Some UI elements and features are still being polished.
- Servers are currently located in Europe; players far from Europe may experience higher latency.

For a detailed, up‑to‑date list of known issues and their status, see the [Known issues page]({% link known-issues.md %}).
